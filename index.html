My Projects

Non-prehensile Robotic Manipulation Optimization

Designed and implemented a simulation environment in MuJoCo using Python to optimize trajectory-following accuracy for a sliding object manipulated by one or two actuators on a planar surface. This project involved developing and comparing control strategies, including LQR and Model Predictive Control (MPC), along with optimization algorithms.

Enhancing Lateral Stability Using Advanced ADAS Systems

Leveraged MATLAB, Simulink, LQR, Sliding Mode Control (SMC), and vehicle dynamics principles to develop and simulate Linear Quadratic Regulator (LQR) and Sliding Mode Controllers (SMC). These controllers were designed to accurately track reference steering input signals for autonomous vehicles. The project utilized detailed vehicle dynamics and wheel error dynamics models.
